# Portfolio { Alfie Pearce }

Personal portfolio website built with Astro

## About

This site showcases my projects, technical skills, and work experience. It is a central place to document software, game development, simulation and programming projects.

## Built with

- Astro, npm
- HTML, CSS, TypeScript
- Neovim

## Features

- Data-diven project system
- Responsive Layout
- Project categories:
    Pinned
    Recent
    Early
- Image and video project previews
- Skills and tooling overview
- Work experience section

## Local Development

Install Dependencies:  

```
npm install
```

Run the development server:

```
npm run dev
```

## Project Structure:

```
src/
├── components/
├── data/
├── layouts/
├── pages/
└── styles/
public/
└── media/
```
